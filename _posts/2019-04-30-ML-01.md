In Lesson 01, I studied basic things of ML(Machine Learning).

First thing is Supervised Learing .
![Supervised Learning](https://t1.daumcdn.net/cfile/tistory/996ABA345B8936D629)

It is that let computer learn with examples which has labels.

Like Alphago, for example, we make the input data sets and output data sets. We let computer know that what we input this examples and what answers when we input that examples.

Second thing is Unsupervised Learning.

It is contrary concept of supervised learning.

Unlike supervised learning, We don't make labels. And that makes computer doesn't know what is inputs and what is outputs.

We can see this examples in google news grouping and word clustering.

Third thing is regression.

When we predict exam's score, we have to predict between 0 to 100 and it should be continous. And it is regression.

Fourth thing is classification.

Unlike regression, we sometimes predict not continous result, but uncontinous result. And when we predict the uncontinous one, we call it classification.

And even the inside of classification, we can classify the binary classification and multi-label classification by the numbers of results.
(binary classification for 2 resluts, multi-label classification for multi results)
